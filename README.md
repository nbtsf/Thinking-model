## 每个Skill包含的内容维度

1. **使用前提**：明确使用该思维模型的前提条件
2. **使用背景**：解释该模型的背景和适用场景
3. **使用条件**：列出具体的应用条件
4. **核心要点**：提炼模型的核心知识点
5. **核心思维**：阐述模型的核心思维方式
6. **核心逻辑**：说明模型的底层逻辑链条
7. **思维链**：详细的思考步骤
8. **如何使用**：具体的应用方法
9. **互联网领域案例**：互联网行业的实际应用案例
10. **Few Shots示例**：对话式示例，展示正确和错误的应用方式

## 使用说明

每个思维模型都是独立的Skill，可直接加载使用。
